# Trendyol Sepet Arayüz Simülasyonu (Java Swing UI)

Bu proje, popüler e-ticaret platformu **Trendyol**'un sepet ve ürün listeleme ekranlarından ilham alınarak hazırlanmış bir **Kullanıcı Arayüzü (GUI)** çalışmasıdır. Projenin temel amacı, Java Swing bileşenlerini kullanarak karmaşık bir alışveriş sayfasını masaüstü ortamında görselleştirmektir.

> [!IMPORTANT]
> **Not:** Bu proje şu aşamada **yalnızca arayüz (UI)** tasarımıdır. Arka plan fonksiyonları (fiyat hesaplama, veritabanı bağlantısı vb.) henüz aktif değildir; görsel yerleşim ve bileşen hiyerarşisi üzerine odaklanılmıştır.

## 🚀 Öne Çıkan Özellikler
* **Bileşen Hiyerarşisi:** `JFrame`, `JPanel`, `JButton`, `JLabel` ve `JTextField` gibi Java Swing öğelerinin düzenli kullanımı.
* **Vurgulu Alanlar:** "Kargo Bedava", "Sepeti Onayla" ve ürün adet seçimi gibi kullanıcıyı yönlendiren görsel elementler.

## 🛠️ Geliştirme Ortamı
Proje aşağıdaki teknik altyapı kullanılarak geliştirilmiştir:

* **İşletim Sistemi:** Windows 11
* **Geliştirme Ortamı (IDE):** Apache NetBeans IDE 20
* **Dil:** Java (JDK 17 veya üzeri önerilir)
* **Kütüphane:** Java Swing & AWT

## 💻 Kullanım Şekli
Projeyi kendi yerel makinenizde görüntülemek için:

1. Bu depoyu (repository) klonlayın veya `.zip` olarak indirin.
2. **Apache NetBeans** editörünü açın.
3. `File -> Open Project` yolunu izleyerek proje klasörünü seçin.
4. Ana dosyayı (Main Class) belirledikten sonra `F6` tuşuna basarak veya `Run` butonuna tıklayarak arayüzü çalıştırın.
